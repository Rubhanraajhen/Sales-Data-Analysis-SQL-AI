Sales Data Analysis using SQL and AI

Objective:
To analyze sales data and extract basic business insights using SQL queries.

Dataset:
The dataset contains sales transaction details such as product, category,
quantity, price, order date, and region.

Tools Used:
- Microsoft Excel
- SQL
- AI tools for explanation and insights

Analysis Performed:
- Total sales calculation
- Category-wise revenue analysis
- Top selling products
- Region-wise sales performance

Key Insights:
- Electronics category contributes the highest revenue.
- Certain products have consistently higher sales volume.
- South and North regions show strong sales performance.
